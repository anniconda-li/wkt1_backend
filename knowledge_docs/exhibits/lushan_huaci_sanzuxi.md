# 黑釉蓝斑花口三足洗

文物ID：lushan_huaci_sanzuxi  
标准名称：黑釉蓝斑花口三足洗  
别名：鲁山花瓷、花瓷、黑釉蓝斑三足洗  
类别：陶瓷

## 视觉检索描述

该文物为一件黑釉蓝斑花口三足洗，整体呈敞口弧腹造型，口沿不规则呈花瓣状。器身主体为深黑色釉面，表面带有自然流淌的蓝色斑纹，釉色光亮且富有层次感。底部有三个兽首形足，造型古朴，腿部细节雕刻清晰，呈现出金属质感。器物内外壁可见釉料垂流和窑变效果，边缘部分露胎，显现出胎体的灰褐色质地。

## 视觉检索关键词

## 导游讲解

黑釉蓝斑花口三足洗属于鲁山花瓷系统，黑釉与蓝斑对比明显，花口和三足造型便于视觉识别。鲁山花瓷以釉色变化和斑块装饰见长，是唐代河南陶瓷面貌的重要代表。
