# 邓公簋

文物ID：denggong_gui  
标准名称：邓公簋  
别名：西周邓公簋、青铜簋  
类别：青铜器

## 视觉检索描述

该文物为一件青铜材质的古代容器，整体呈圆形，具有盖子和底座。器身由多层横向凸起的弦纹构成，表面覆盖着绿色的铜锈，显示出年代感。盖顶中央有一个圆形凸起，边缘有细致的几何纹饰。器身两侧各有一个兽首形环耳，耳部造型生动，带有环状把手。盖与器身连接处及底部均有复杂的云雷纹或卷曲纹样装饰，纹路清晰且富有层次感。整体造型庄重，工艺精湛，体现了古代青铜器的典型特征。

## 导游讲解

邓公簋是西周青铜簋，1984年出土于平顶山应国墓地。簋原为盛放熟食的食器，在墓葬和礼制语境中也体现贵族身份与婚姻关系，是研究应国与邓国关系的重要青铜器。
