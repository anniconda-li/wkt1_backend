# 应国玉鹰

文物ID：yingguo_yuying  
标准名称：应国玉鹰  
别名：白玉线雕鹰、西周白玉线雕鹰、玉鹰  
类别：玉器

## 视觉检索描述

该文物为一件玉质雕刻品，整体呈鹰的形态，造型简洁而富有动感。鹰的双翼展开，边缘圆润，尾部微微上翘。身体部分雕刻有流畅的线条，表现出羽毛的层次感和纹理细节。头部略微抬起，喙部清晰可见，眼睛位置未明确刻画。材质为浅黄色至米白色玉石，表面光滑莹润，局部带有自然的褐色沁色，尤其在鹰的背部和尾部较为明显。整体雕刻工艺精细，线条流畅，具有典型的古代玉器风格。

## 视觉检索关键词

玉器、鹰形、双翼展开、线雕、羽毛纹理、头部微抬、喙部清晰、褐色沁色、光滑莹润、古代风格

## 文物基础信息

年代：西周晚期  
用途：佩器或身份装饰  
材质：白玉  
馆藏：平顶山市博物馆  
出土信息：1986年出土于平顶山应国墓地

## 导游讲解

应国玉鹰又称白玉线雕鹰，是平顶山市博物馆的重要玉器。它出土于1986年的平顶山应国墓地，整体作展翼鹰形，线刻线条清晰流畅，是理解应国文化和鹰城地名文化的重要展品。
