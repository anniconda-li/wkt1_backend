# 束腰垂鳞纹升鼎

文物ID：shuyao_chuilinwen_shengding  
标准名称：束腰垂鳞纹升鼎  
别名：升鼎、青铜鼎  
类别：青铜器

## 视觉检索描述

这是一件束腰垂鳞纹升鼎，整体呈圆形敞口，腹部向内收束形成束腰状，底部由三足支撑。器物表面覆盖着深绿色的铜锈，显现出青铜材质的古朴质感。腹部饰有规则排列的垂鳞纹，纹路细腻且富有层次感。两侧各有一个向外弯曲的环形耳，耳部边缘圆润。器腹中部对称装饰有两个兽首形环耳，下方悬挂一个环形提梁，提梁上亦有精细纹饰。三足为粗壮的兽蹄形，底部略宽，与器身连接处有凸起的装饰带。整体造型庄重典雅，细节丰富。
