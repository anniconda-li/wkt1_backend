# 未知问题与边界回答

## 实时开放状态

正文事实：
开放状态可能因节假日、维护、活动而变化。常规时间可说周二至周日九点到十七点、周一闭馆，但当天是否开放必须以最新公告为准。

可能问题：
- 今天开门吗？
- 现在能进去吗？
- 今天闭馆吗？

推荐回答：
实时开放状态可能变化，请以博物馆最新公告为准。

## 预约政策

正文事实：
预约要求可能随客流和活动变化，不应固定回答需要或不需要。

可能问题：
- 要预约吗？
- 团队要预约吗？
- 今天能直接进吗？

推荐回答：
预约要求可能变化，请以博物馆最新公告为准。

## 票价门票

正文事实：
门票和入馆政策可能调整，知识库不编造收费、免费和优惠细节。

可能问题：
- 门票多少钱？
- 免费吗？
- 学生有优惠吗？

推荐回答：
票价和入馆政策请以博物馆最新公告为准。

## 停车餐饮

正文事实：
停车位、餐饮点、营业状态和收费都属于实时服务信息，建议使用现场提示或实时平台。

可能问题：
- 停车方便吗？
- 附近吃饭吗？
- 馆里有餐厅吗？

推荐回答：
这类信息变化较快，请看现场提示或实时平台。

## 洗手间和无障碍

正文事实：
洗手间、无障碍设施、休息区等具体位置和状态，应以现场导览标识或工作人员说明为准。

可能问题：
- 洗手间在哪？
- 有轮椅吗？
- 老人方便吗？

推荐回答：
请查看现场标识，或询问工作人员。

## 具体展品位置

正文事实：
展品陈列可能调整，不能编造楼层、展柜号或具体位置。

可能问题：
- 玉鹰在哪个展柜？
- 几楼能看到？
- 展品具体在哪里？

推荐回答：
具体展品位置请以现场展陈为准。

## 不存在或未查到展品

正文事实：
用户问到知识库没有记录的具体展品时，不应猜测存在。

可能问题：
- 有没有某某文物？
- 这个展品是真的吗？
- 某件东西在哪？

推荐回答：
这个问题我暂时没有查到相关资料。

## 非本馆问题

正文事实：
天气、酒店、饭店、金融、医疗、政治等内容不属于本馆导游知识库范围。

可能问题：
- 天气怎么样？
- 附近酒店推荐？
- 能给医疗建议吗？

推荐回答：
这个问题超出当前导游知识库范围。
