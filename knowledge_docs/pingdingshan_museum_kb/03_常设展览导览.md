# 常设展览导览

## 鹰城古韵总览

正文事实：
平顶山博物馆基本陈列包括“鹰城古韵：平顶山历史与文化陈列”，可按山下故原、应国印象、楚汉文明、唐宋遗韵、鹰城记忆等线索理解。这不是单纯罗列文物，而是把平顶山从早期文明、应国记忆、区域历史、地方审美一路讲到现代城市。

可能问题：
- 鹰城古韵讲什么？
- 常设展览怎么看？
- 这个馆主线是什么？

推荐回答：
鹰城古韵讲平顶山从古到今的历史文化主线。

## 山下故原

正文事实：
山下故原适合作为开头理解。它强调平顶山这片土地早期的历史文化基础，让游客知道后面的应国、楚汉、唐宋并不是突然出现的。具体展品以现场陈列为准。

可能问题：
- 山下故原讲什么？
- 第一个展厅看什么？
- 早期文明怎么看？

推荐回答：
山下故原讲平顶山早期文化基础。

## 应国印象

正文事实：
应国印象是本馆核心展览线索之一。平顶山一带有重要应国历史记忆，应国玉鹰出土于平顶山应国墓地，鹰城称号也常从这里讲起。

可能问题：
- 应国印象讲什么？
- 应国展厅重要吗？
- 应国和鹰城有关吗？

推荐回答：
应国印象是理解应国文化和鹰城来源的重点。

## 楚汉文明

正文事实：
楚汉文明让游客把平顶山放进中原历史变动中看。这里不是孤立地方，而是在楚汉以来的区域交流、政权变化和社会发展中延续文化。

可能问题：
- 楚汉文明讲什么？
- 楚汉和平顶山有什么关系？
- 这个展厅怎么听？

推荐回答：
楚汉文明帮助理解平顶山和中原历史的联系。

## 唐宋遗韵

正文事实：
唐宋遗韵适合看地方文化的继续发展。游客可以从生活器物、审美变化、地方工艺和历史延续角度理解，不要只追问单件展品值多少钱。

可能问题：
- 唐宋遗韵讲什么？
- 唐宋展厅怎么看？
- 这里有什么意义？

推荐回答：
唐宋遗韵讲地方文化的延续和审美变化。

## 鹰城记忆

正文事实：
鹰城记忆把展览拉回现代城市。它适合本地游客找熟悉感，也适合外地游客理解平顶山如何从古代文化线索走到今天的城市认同。

可能问题：
- 鹰城记忆讲什么？
- 有没有现代内容？
- 本地人看什么？

推荐回答：
鹰城记忆让古代历史和今天的平顶山接上。

## 第一次参观顺序

正文事实：
第一次参观建议先听鹰城古韵总览，再看应国印象和应国玉鹰，然后按时间看楚汉文明、唐宋遗韵和鹰城记忆。时间少就先抓应国文化。

可能问题：
- 第一次怎么看？
- 先看哪个展厅？
- 不懂历史怎么走？

推荐回答：
第一次来先抓应国印象、应国玉鹰和鹰城记忆。

## 只看一个展厅

正文事实：
如果只能看一个展厅，建议优先选择应国印象。因为它最能解释应国玉鹰、应国文化和鹰城称号之间的关系，是本馆识别度最高的主线。

可能问题：
- 只看一个看哪个？
- 赶时间看什么？
- 哪个展厅最重要？

推荐回答：
只看一个，建议看应国印象。

## 带孩子看展厅

正文事实：
带孩子参观时，不必讲太多断代术语。可以让孩子找“像书的建筑”“鹰形玉器”“鹰城名字”，用可观察的形象带出历史故事。

可能问题：
- 带孩子看展厅怎么讲？
- 孩子听不懂怎么办？
- 亲子看什么？

推荐回答：
带孩子抓形状、故事和城市名字。

## 外地游客看展厅

正文事实：
外地游客可以把展览当作平顶山说明书：先明白应国文化，再看应国玉鹰，最后理解为什么这座城市被称为鹰城。

可能问题：
- 外地人怎么看？
- 第一次来平顶山看什么？
- 展览有什么用？

推荐回答：
外地游客抓应国、玉鹰、鹰城三点就能看懂主线。
