# 推荐参观路线

## 十五分钟极简路线

正文事实：
先看建筑竹简意象，再听应国玉鹰的核心信息，最后用一句话理解鹰城。十五分钟不要追求完整展览，只抓城市符号。

可能问题：
- 十五分钟怎么看？
- 我赶时间看什么？
- 只看重点怎么走？

推荐回答：
十五分钟抓竹简建筑、应国玉鹰和鹰城来源。要不要我把它换成更短的边走边听版？

## 三十分钟快速路线

正文事实：
三十分钟建议先听鹰城古韵总览，再看应国印象和应国玉鹰，最后快速了解鹰城记忆。青铜器、陶瓷等内容可作为补充。

可能问题：
- 三十分钟怎么看？
- 半小时够吗？
- 快速路线怎么走？

推荐回答：
三十分钟重点看应国玉鹰、应国文化和鹰城记忆。如果带孩子，我也可以换成亲子路线。

## 一小时标准路线

正文事实：
一小时可按山下故原、应国印象、楚汉文明、唐宋遗韵、鹰城记忆的顺序理解。重点不是走到具体展柜，而是听懂从早期到现代的主线。

可能问题：
- 一小时怎么看？
- 标准路线是什么？
- 第一次怎么安排？

推荐回答：
一小时按展览主线走，能看清平顶山历史脉络。需不需要我帮你从当前展厅开始调整顺序？

## 九十分钟深度路线

正文事实：
九十分钟以上适合细看展板和文物类型。建议重点停留在应国印象和应国玉鹰，再补充楚汉、唐宋、陶瓷和城市记忆。

可能问题：
- 深度参观怎么走？
- 九十分钟看什么？
- 慢慢看怎么安排？

推荐回答：
九十分钟可慢看应国文化、玉鹰、陶瓷和城市记忆。

## 亲子路线

正文事实：
亲子路线不要用太多术语。先看像竹简的建筑，再看应国玉鹰，再讲平顶山为什么叫鹰城，最后让孩子找自己记住的城市关键词。

可能问题：
- 亲子路线怎么走？
- 带孩子看什么？
- 孩子听不懂怎么办？

推荐回答：
亲子路线抓建筑、玉鹰和鹰城故事。

## 外地游客路线

正文事实：
外地游客先理解城市：平顶山不是只有现代印象，也有应国文化和鹰城记忆。先听应国玉鹰，再看鹰城古韵主线。

可能问题：
- 外地游客怎么看？
- 第一次来平顶山看什么？
- 外地人怎么理解？

推荐回答：
外地游客先看应国玉鹰，再听鹰城文化。

## 本地游客路线

正文事实：
本地游客可从鹰城记忆找熟悉感，再回到应国玉鹰和应国文化，理解熟悉城市名字背后的历史来源。

可能问题：
- 本地人怎么看？
- 本地游客有什么看头？
- 鹰城记忆怎么听？

推荐回答：
本地人可从鹰城记忆回看应国文化。

## 只看重点文物路线

正文事实：
只看重点文物，就围绕应国玉鹰展开：先知道它也叫白玉线雕鹰，再听出土地点、年代背景和鹰城关系。

可能问题：
- 只看文物看什么？
- 重点文物路线怎么走？
- 最值得看的文物是哪件？

推荐回答：
只看重点文物，就围绕应国玉鹰来听。

## 鹰城主题路线

正文事实：
想了解鹰城，先听应国文化，再听应国玉鹰，最后看鹰城记忆。这样能从古代应国讲到现代城市称号。

可能问题：
- 鹰城主题怎么看？
- 为什么叫鹰城？
- 鹰城路线怎么走？

推荐回答：
了解鹰城，重点听应国文化、玉鹰和鹰城记忆。

## 陶瓷文化路线

正文事实：
想看陶瓷文化，可关注鲁山花瓷、郏县钧瓷等地方线索。具体展品以现场为准，导览重点放在工艺、釉色和地方审美。

可能问题：
- 陶瓷文化怎么看？
- 鲁山花瓷在哪看？
- 郏县钧瓷怎么看？

推荐回答：
看陶瓷，关注鲁山花瓷和郏县钧瓷线索。
